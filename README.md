# Sports Betting Bot

An advanced sports betting bot that uses machine learning to predict football match outcomes and create optimized parleys.

## Features

- **Real-time Odds Fetching**: Integrates with The Odds API for current betting odds
- **Historical Data Analysis**: Uses API-Football for 2+ years of historical data
- **Machine Learning**: Gradient Boosting models for prediction
- **Parley Management**: Automatically creates 5 optimized parleys daily
- **Performance Tracking**: Stores and analyzes betting history
- **Continuous Improvement**: Weekly model retraining for better predictions
- **Web Dashboard**: Real-time monitoring and statistics
- **Multiple Bet Types**: Head-to-head, spreads, totals, BTTS, corners, etc.
